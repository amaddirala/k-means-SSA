This MATLAB script compares a proposed K-means SSA-based method with several existing techniques for removing eye-blink (EOG) artifacts from single-channel EEG recordings. 
A simulated EEG signal contaminated with EOG artifacts is used to evaluate the ability of each method to estimate the artifact and reconstruct the underlying clean EEG signal.
